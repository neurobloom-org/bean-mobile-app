## 🌟 Overview
**Bean** is more than an app; it is the **central command center** for the Bean Mental Wellness Robot. Developed by **NeuroBloom**, this mobile interface bridges the gap between physical robotics and cloud-based emotional intelligence. It serves as a secure, empathetic bridge connecting users, guardians, and therapeutic systems into a single supportive ecosystem.

---

## 🛠 Technical Architecture
The Bean Mobile App is built on a **High-Availability "Thin-Client" Architecture**:
* **Frontend:** React Native (Cross-platform iOS/Android)
* **Backend:** Flask Micro-framework (Hosted on Render)
* **Database & Auth:** Supabase (PostgreSQL with Row Level Security)
* **Intelligence:** Retrieval-Augmented Generation (RAG) via Microsoft Azure OpenAI
* **Connectivity:** Bluetooth Low Energy (BLE) & WiFi Handshake for Robot provisioning

---

## 👤 User Ecosystem

### 🧘 **Primary User (The Companion)**
* **Real-time Interaction:** Engage in empathetic dialogue with the Bean Robot.
* **Emotional Regulation:** Access grounding exercises and mood-tracking tools.
* **Routine Mastery:** Managed task lists and focus sessions to reduce cognitive load.

### 🛡️ **Guardian / Therapist (The Support)**
* **Progress Analytics:** View high-level emotional trends via the Mood Insights dashboard.
* **Safety Protocols:** Receive immediate **SOS Alerts** during high-anxiety episodes.
* **Intervention Tools:** Remotely assign tasks or wellness activities to support the user’s journey.

---

## 🚀 Key Features

* **📊 Emotional Intelligence Analytics:** Visualize mood fluctuations over time using data-driven insights.
* **🤖 Robot Synchronization:** One-touch pairing via a secure BLE-to-WiFi handshake.
* **🔐 Clinical-Grade Security:** Role-based access control (RBAC) ensuring patient data remains private.
* **🧘 Mindfulness Hub:** Integrated focus timers and calming audio activities.
* **✅ Collaborative Task Management:** Real-time task syncing between the guardian app and the robot’s voice interface.
